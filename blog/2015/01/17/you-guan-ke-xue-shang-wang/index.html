Goagent

goagent 更新版，设置和旧版本略有不同，不仅要修改proxy.ini 里面的appid，还要找到[dns]项，将enable设置为1。 然后uploader，重新上传一次配置。（不想重新上传也有办法，见ps） 如果还是出现类似于 The requested URL /_gh/ was not found on this server. 的问题，Don’t panic! 在浏览器里重新导入一次CA.crt证书即可，local文件夹下。

pac篇 在实验室主机那里，直接把proxy.pac文件修改 主要是替换127.0.0.1为主机ip 把FindProxyForURL 中的defaultproxy 变量由原来的’DIRECT’ 修改为’PROXY host_ip:port’ FindProxyForURLByAdblock中的’DIRECT’ 修改为’PROXY host_ip:port’ FindProxyForURLByAutoProxy中的’DIRECT’ 修改为’PROXY host_ip:port’ 其中，host_ip:port为所需的实验室主机ip及其端口 新建pac情景模式，复制进去即可

实验室主机那边，需要修改proxy.ini文件， [listen]里的ip项和[dns]里的listen项都从原本默认的127.0.0.1改为0.0.0.0

Enjoy it!

ps: 在google code issue 里面看到有位热心的同学贡献了一堆appid，你懂的！！
